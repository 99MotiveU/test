4. 실시간 면접 대기 및 순번 호출 플랫폼

프로젝트명: INTV-WAIT (인터뷰-웨잇)

🎯 개요

비대면/온라인 면접 또는 박람회에서 실시간 면접 대기열과 순번을 자동 관리하고, 면접자가 본인의 순서를 실시간으로 확인하고 입장할 수 있게 하는 실시간 대기열 플랫폼. 여러 면접장이 동시간대에 운영되더라도 안정적인 분산 처리와 대기열 운영이 가능하도록 MSA 기반 구조로 확장.

🧩 기술 도전 포인트

Redis List 또는 Sorted Set을 이용한 대기열 큐 설계

WebSocket 기반 실시간 순번 알림

면접장별로 분리된 큐/세션 구성

관리자 패널을 통한 수동/자동 순번 호출

MSA 구조에서 면접장 단위 서비스 분리 및 확장 가능

🔧 기술 스택

Spring Boot, Redis, WebSocket

PostgreSQL, Docker, Azure, React (프론트엔드)

서비스 구조: 면접장 서비스 / 대기열 서비스 / 알림 서비스

📌 주요 기능

관리자: 면접장 등록, 대기열 설정, 수동/자동 호출 제어

참가자: 대기 등록 → 예상 순서 확인 → 실시간 호출 수신

실시간 브로드캐스트 및 모바일 대응 UI

🌟 가치

MSA 기반 실시간 대기열 플랫폼 설계 경험

실시간 유저 상태 동기화 및 확장 가능한 구조 구축

실제 기업 면접 또는 박람회에서 적용 가능한 실용성
